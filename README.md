# Test Parsing Task

Parsing product data by Requests / BS4, Selenium and Playwright

## Set Environment Variable

Create env.sh from env.example.sh and run it:

```terminaloutput
    source env.sh
```

## Run

1. Create ProductInfo instances `1_get_links.py`
2. Requests / BS4 `2_get_info_data.py`
3. Selenium `3_get_info_selenium.py`
4. Playwright `4_get_info_playwright.py`
